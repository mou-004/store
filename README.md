# FakeStore Product Explorer

A polished responsive React application that fetches products from `https://fakestoreapi.com/products`.

## Features

- Modern responsive UI for mobile, tablet and desktop
- Product listing with image, title, price, category and rating
- Debounced product search
- Category filter with clean tab/pill design
- Sorting by name, price and rating
- Product details modal
- Loading skeleton
- Error handling
- Pagination
- Dark/light theme using Context API
- Clean reusable component structure

## Folder Structure

```text
src/
├── components/

```

## Run the Project

```bash
npm install

```
